About HDI Predictor

The Human Development Index (HDI) Predictor is a Machine Learning web application that predicts the HDI score of a country using important socio-economic indicators.

Features Used

  • Life Expectancy at Birth
  • Expected Years of Education
  • Mean Years of Education
  • Gross National Income (GNI) per Capita

Machine Learning Model

Linear Regression (Scikit-learn)

Technologies Used

  • Python
  • Flask
  • Pandas
  • NumPy
  • Matplotlib
  • Seaborn
  • Scikit-learn
  • HTML
  • CSS

Dataset

Human Development Index (HDI) Dataset (UNDP/Kaggle)

Outcome

This application predicts the HDI score and classifies it as Very High, High, Medium, or Low Human Development.
