# Human Development Index (HDI) Predictor

## Overview
This project predicts the Human Development Index (HDI) using Machine Learning.

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- HTML
- CSS

## Features
- Predict HDI Score
- Classify development level
- User-friendly web interface

## Machine Learning Model
Linear Regression

## How to Run

```bash
pip install -r requirements.txt
python model.py
python app.py
```

Open:

```
http://127.0.0.1:5000
```
